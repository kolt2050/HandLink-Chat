Group HandLink Chat

Manual encrypted P2P group chat for Chrome. Create an invite, exchange the answer manually, and chat directly through WebRTC without a project-owned backend server.

Privacy Policy Chrome Web Store copy

What It Does

  • Creates encrypted manual WebRTC chat rooms.
  • Supports multiple participants, mentions, images, clickable links, and local history.
  • Stores chat data locally in the browser and exchanges messages only with invited peers.
